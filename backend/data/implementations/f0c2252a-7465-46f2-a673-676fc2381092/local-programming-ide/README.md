# Local Programming IDE

## Overview
A local AI-powered Integrated Development Environment (IDE) that provides a chat-like experience for programming across various languages. This project is built using Electron and React to ensure cross-platform compatibility.

## Features
- Multi-language support through plugins.
- Chat-like interaction with AI agents.
- Utilizes available hardware resources efficiently (GPU, NPU, CPU).

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/local-programming-ide.git
   ```

2. Navigate to the project directory:
   ```
   cd local-programming-ide
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the application:
   ```
   npm start
   ```

## Development

To build and run the IDE in development mode, use:

```
npm start
```

For production builds, use:

```
npm run build
```

## Security

All changes are contained within the folder from which the app is executed. Strict file access controls ensure security.
